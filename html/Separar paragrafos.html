Tanto compiladores quanto interpretadores são usados para transformar um programa escrito em uma linguagem de alto nível em código de máquina compreendido por computadores. Porém, há algumas diferenças entre eles.

Sobre essas diferenças, tem-se que:

  1. Um interpretador necessita sempre ler o código fonte original para executá-lo.
  2. Com o uso de compliadores, a execução do programa é separada da compilação. É executado somente depois que todo o programa de saída é compilado
  3. O compilador lê uma única instrução e mostra o erro, se houver. Você deve corrigir o erro para seguir para a próxima linha.
  4. Com o uso de interpretadores, a execução do programa faz parte do processo de interpretação, portanto, é realizada linha por linha

É correto apenas o que se afirma em:

  1. I e II
  2. I, II e IV
  3. II e III
  4. I, III e IV
  5. I, II e III
